Week 6: 4/23 & 4/25 2002

The Web Based Portfolio

This will be the final project of the semester.

Ideally, we'll all be done by May 14th and we can all go home early.

We will work on this project in class for the remainder of the semester. You should continue work on this project outside of class.

The Assignment

You will create a small portfolio web site from templates I provide.

The web site will include:

  • A Home page,including:
    • 6 Thumbnail images, linking to the portfolio pages.
    • A 300px x 350px graphic/logo to identify your site
    • Your name and email address
  • 6 Portfolio pages highlighting your work, including:
    • A large portfolio image (no bigger than 600 px wide or tall)
    • A breif written description about the work shown in the image
    • A 700px x 100px header graphic for identifying your website
  • A Resmé page

Due Dates/Grading

Your website should be completed and uploaded to your UNCG server space by the end of the day on May 14th.

The portfolio website will be graded in two areas.

  • Completion of components (A website with all components listed above would receive a "100")
  • Overall aesthetic/design quality (A website displaying competent, attractive, communicative graphics with a well-chosen typeface would receive a better grade than a website which displays the default fonts and dull graphics)

Steps To Completion

You should have already created your portfolio images, and thumbnails.

What remains is to:

  1. Create two website graphics
  2. Create a resumé in HTML
  3. Customize the HTML templates (insert your images, adjust colors, typefaces, etc)
  4. Upload everything to your web server space
